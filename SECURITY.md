# Security Policy

## Supported Versions

Most of FEIBox's versions follow a lifecycle. After that,it would be no longer supported

| Version | Supported           |
| ------- | ------------------- |
| 0.8.1dev| WIP |
| 0.8.0   | Currently supported |
| 0.8.0dev| EOL |
| 0.7.0   | Currently supported |
| 0.7.0dev| EOL |
| 0.6.0-  | EOL |
